Small experimental Jupyter Notebook project that provides an interactive world map for placing travel pins, assigning statuses, and storing them locally in a JSON file.

Built for fun and exploration of ipyleaflet and ipywidgets.

---

## Features

- Click on map to add pins  
- Edit pin name and status  
- Status colors:
  - Visited (green)
  - Want to visit (blue)
  - Not interested (red)
- Marker clustering  
- Local JSON storage
